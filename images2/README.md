# Recursos fotográficos (excluidos del repositorio)

Para evitar los errores al generar solicitudes de extracción que incluyen imágenes binarias, este repositorio solo conserva las rutas de referencia utilizadas en el sitio.

Coloca manualmente en este directorio los archivos fotográficos optimizados con los mismos nombres mencionados en el código HTML/CSS. Los formatos recomendados son **JPG** para fotografías y **PNG** para gráficos con transparencia.

Asegúrate de mantener tamaños moderados (entre 1200 y 1600 px por el lado mayor) y comprimirlos para la web antes de subirlos a producción.
